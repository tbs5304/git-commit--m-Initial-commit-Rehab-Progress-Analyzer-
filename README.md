# Rehab Progress Analyzer

リハビリ患者の進捗データを分析するPythonツールです。

## 概要
患者データから進捗状況を分析し、
セラピスト別・患者別のサマリーを自動生成します。

## 使用技術
- Python
- pandas
- CSVデータ分析

## 実行方法
```bash
pip install -r requirements.txt
python rehab_progress_analyzer.py

## 背景

医療現場では患者進捗データの集計が手作業で行われることが多く、
分析や意思決定に時間を要する課題があります。

本ツールは、CSV形式の患者データから
進捗状況を自動集計・可視化することで、
業務効率化とデータ活用を目的として開発しました。

## 解決できる課題

- 手作業による集計時間の削減
- セラピスト別成果の可視化
- 疾患別改善傾向の把握
- データに基づく意思決定支援

## 出力内容

- 患者別進捗データ
- セラピスト別サマリー
- 疾患別改善率サマリー
